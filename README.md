# RAG Lab

Experimental workbench for building, comparing, evaluating, and documenting RAG pipelines over client document datasets.

The project is not a generic chat-with-PDF demo. Its purpose is to run controlled experiments and export production-ready RAG recipes.

Core flow:

```text
dataset → loading → chunking → indexing → retrieval → answer → evaluation → report → recipe
```
